# Deva Matha College Website

Welcome to the Deva Matha College (Arts & Science, Paisakary, Kannur) official website repository.

## About the Project
This project is a modern web application built for Deva Matha College, established in 2003 and affiliated to the University of Kannur. It offers a variety of UG and PG courses including BCA, BBA, BCom, BA, BSc, MA, and MCom.

## Technologies Used
This project is built using modern web development tools:
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

### Prerequisites
- Node.js installed

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate into the directory:
   ```sh
   cd deva-matha-renewed-main
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

### Running Locally
To start the development server:
```sh
npm run dev
```

### Building for Production
To build the application:
```sh
npm run build
```
